# 🏀 Análisis de Performance NBA 2015-16

## Resumen
Este proyecto utiliza **Python** para analizar el rendimiento de jugadores y agruparlos según su eficiencia operativa mediante técnicas de **Clustering**.

## Tecnologías Utilizadas
* **Pandas & Numpy:** Limpieza de datos y feature engineering.
* **Seaborn & Matplotlib:** Visualización de clusters y mapas de calor.
* **Scikit-learn:** Implementación de modelos **KMeans** y **DBSCAN**.
* **Google Colab:** Entorno de desarrollo.

## Hallazgos Clave
* Identificación de **3 perfiles principales** de jugadores según su impacto en el juego.
* Detección de **outliers** de alta eficiencia que superan significativamente el promedio de la liga.
